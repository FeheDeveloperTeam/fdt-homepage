# 로고 / 아이콘 폴더

이 폴더에 FDT(FeheDeveloperTeam) 팀 로고, 파비콘용 이미지 등을 넣어주세요.

권장 파일:
- `logo.svg` 또는 `logo.png` — 헤더에 표시될 메인 로고 ([Header.jsx](../../../components/Header/Header.jsx)에서 사용)
- `logo-white.svg` — 어두운 배경(푸터 등)용 흰색 버전 (선택)
- `favicon.svg` / `favicon.ico` — [public/favicon.svg](../../../../public/favicon.svg) 교체용, `public/` 폴더에 직접 넣으면 됩니다

파일을 넣은 후 [Header.jsx](../../../components/Header/Header.jsx)와 [Footer.jsx](../../../components/Footer/Footer.jsx)에서 import 경로를 연결해 드릴게요.
